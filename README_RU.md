## О проекте

LineExtender демонстрирует практическое применение следующих технологий:
- Захват экрана в реальном времени через Windows GDI
- Обнаружение линий с использованием преобразования Хафа и OpenCV
- Отображение полупрозрачного overlay с помощью Pygame
- Динамическая настройка параметров через графический интерфейс

## Установка

**Требования**:
- Windows 10/11
- Python 3.8 или новее

```bash
git clone https://github.com/aidiors/LineExtender.git
cd LineExtender
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

## Использование

1. Запустите приложение с указанием названия целевого окна:
```bash
python src/main.py --window "Название приложения"
```

2. Управление:
   - Нажмите клавишу `INSERT` для завершения работы приложения
   - Настройте параметры обнаружения с помощью панели управления OpenCV

## Конфигурация

Все параметры можно настроить во время выполнения через панель управления OpenCV:

| Параметр          | Описание                                      | Допустимый диапазон |
|--------------------|-----------------------------------------------|---------------------|
| Размер захвата     | Размер области вокруг курсора (в пикселях)    | 120-800             |
| Порог Хафа         | Чувствительность обнаружения линий            | 1-140               |
| Мин. длина линии   | Минимальная длина сегмента линии для учета    | 1-200               |
| Макс. разрыв линии | Максимальный разрыв между сегментами для слияния | 1-100            |

Значения по умолчанию оптимизированы для общих случаев использования. Сохраните оптимальные настройки, изменив файл `constants.py`.
